e-plantShopping is a React-based web application that allows users to browse different categories of plants and add them to a shopping cart. The application provides a simple and user-friendly shopping experience for purchasing indoor and outdoor plants online.

## Features

- Browse plants by category (Indoor Plants, Outdoor Plants, etc.)
- View plant name, image, and price
- Add plants to the shopping cart
- Increase or decrease the quantity of items in the cart
- Remove items from the cart
- Automatically calculate the total price of items in the cart
- Navigate between the product listing and cart pages using a navigation bar

## Technologies Used

- React
- JavaScript
- HTML
- CSS
